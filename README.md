# 🏥 HospitaKr Enterprise Pro
**A Modern Hospital Management System built with Python & CustomTkinter.**

HospitaKr Enterprise is a comprehensive desktop application designed to streamline hospital operations, including patient registration, staff management, and clinical reporting.



---

## ✨ Key Features
* **🔐 Secure Authentication:** Integrated Login/Signup system for administrative access.
* **📊 Real-time Analytics:** Dashboard showing total patients, hospital personnel, and net revenue.
* **👥 Patient Management:** Full CRUD (Create, Read, Update, Delete) functionality for patient records.
* **👨‍⚕️ Staff Directory:** Dedicated sections for Doctors and Support Staff with specialty tracking.
* **📄 Clinical Reports:** Automated **PDF generation** for medical records using ReportLab.
* **🎨 Modern UI:** Sleek, dark-themed interface powered by CustomTkinter.

---

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **GUI Library:** CustomTkinter
* **Database:** SQLite3
* **Reporting:** ReportLab (PDF Engine)
